# OpenApiV3Parser

A lightweight OpenAPI v3 parser for .NET Standard 2.0 that parses OpenAPI specification documents and maps them to strongly-typed C# models.

## Features

- Parse OpenAPI 3.0.x specification documents (JSON format)
- Strongly-typed C# models for OpenAPI components
- Support for paths, operations, schemas, and components
- Built on System.Text.Json for performance
- .NET Standard 2.0 compatible
